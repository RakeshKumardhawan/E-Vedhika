# E-VEDHIKA Agent Instructions

1. **Automatic What's New Updates**: Whenever you add a new feature, make a small update, or change any UI element in the user's application, you MUST automatically add a new entry to the `SYSTEM_UPDATES` array in `src/App.tsx`. Make sure it reflects the latest changes with appropriate version numbers, date, badge, title, and description in Telugu.

2. Always ensure translations or new content is primarily in Telugu where applicable, matching the existing tone of the application.
3. Make sure to adhere to existing Tailwind design patterns and responsive breakpoints.
